# PPTX 播放兼容范围与验收

## 1. 状态

- 状态：第一版范围已确定
- 日期：2026-07-11
- 关联设计：[PPTX 播放正式实现设计](pptx-playback-implementation-design.md)

本文定义“支持”具体指什么。没有列入严格支持的能力，不得在产品文案中笼统称为兼容 PowerPoint 动画。

## 2. 四种处理结果

| 结果 | 含义 | 默认是否执行 |
|---|---|---|
| 严格执行 | 目标、时间、参数、开始和结束状态通过批准素材对照 | 是 |
| 近似执行 | 类型和方向相近，但画面算法不等同 PowerPoint | 仅 `approximation="safe"` |
| 静态保留 | 对象继续显示，不执行该动画 | 是 |
| 无法解析 | 结构损坏或节点未知，记录原始摘要 | 否 |

默认 `approximation="off"`。近似效果必须进入能力报告，不能计入严格执行数量。

## 3. 第一版功能范围

### 3.1 时间与触发

| 功能 | 第一版处理 | 备注 |
|---|---|---|
| 单击时 | 严格执行 | 普通下一步边界 |
| 与上一动画同时 | 严格执行 | 相同触发组开始时间一致 |
| 上一动画之后 | 严格执行 | 前一效果结束后计算延迟 |
| 延迟和持续时间 | 严格执行 | 毫秒模型 |
| 有限重复 | 严格执行 | 必须有结束时间 |
| 自动往返 | 严格执行 | 已通过真实文件验证 |
| 保持和恢复 | 严格执行 | 属性轨道求值 |
| 指定对象点击 | 严格执行 | 先于普通下一步 |
| 媒体书签 | 严格执行 | 正向越过触发，回退后可再次触发 |
| 节点开始和结束引用 | 严格执行批准子集 | 未识别引用静态保留 |
| 无限重复 | 严格执行播放，限制自动换页 | 重置、跳页和销毁必须可终止 |
| 悬停触发 | 静态保留 | 第一版不接入 |

### 3.2 对象动画

| 功能 | 第一版处理 | 目标范围 |
|---|---|---|
| 出现、消失 | 严格执行 | 顶层对象、已标记组合子对象 |
| 淡入、淡出 | 严格执行 | 同上 |
| 擦除 | 严格执行左、右方向 | 其他方向逐项加素材 |
| 缩放 | 严格执行 | 使用原始起止比例 |
| 旋转 | 严格执行 | 使用原始角度 |
| 直线运动 | 严格执行 | 页面坐标换算 |
| 曲线和自定义路径 | 静态保留 | 完整路径换算未验证 |
| 透明度强调 | 严格执行 | 有原始数值时 |
| 缩放和旋转强调 | 严格执行 | 有原始数值时 |
| 颜色强调 | 静态保留 | 探索版只有亮度近似 |
| 闪烁等复杂强调 | 近似或静态 | 默认静态 |
| 同对象不同属性并行 | 严格执行 | 合并属性轨道 |
| 同对象同属性重叠 | 严格执行批准规则 | 必须通过冲突素材 |

### 3.3 文字、图表和 SmartArt

| 功能 | 第一版处理 |
|---|---|
| 整个文字框 | 严格执行 |
| 按段落 | 严格执行，要求段落 DOM 标记 |
| 按词、按字 | 静态保留 |
| 整个图表或 SmartArt | 严格执行为一个对象 |
| 图表按系列、分类或元素 | 静态保留并报告 |
| SmartArt 按节点或层级 | 静态保留并报告 |

### 3.4 页面切换

| 功能 | 第一版处理 |
|---|---|
| 无效果、直接切换 | 严格执行 |
| 淡化 | 严格执行 |
| 推进 | 严格执行已批准方向 |
| 擦除 | 严格执行已批准方向 |
| 其他二维切换 | 近似或静态，默认静态 |
| Morph 强身份对象 | 近似执行，单独标记 |
| Morph 普通名称或文字匹配 | 默认静态 |
| 三维切换 | 近似或静态，默认静态 |
| 切换声音 | 静态保留，后续逐项开放 |

### 3.5 媒体、导航和页面行为

| 功能 | 第一版处理 |
|---|---|
| 嵌入视频播放、暂停、停止 | 严格执行 |
| 嵌入音频播放、暂停、停止 | 严格执行 |
| 循环、音量和裁剪 | 严格执行，正式素材已补齐 |
| 浏览器阻止自动播放 | 显示用户恢复入口 |
| 外部媒体 | 默认禁用 |
| 自动换页 | 严格执行 |
| 普通演示跳过隐藏页 | 严格执行 |
| 指定跳转进入隐藏页 | 严格执行 |
| 内部页面跳转 | 严格执行 |
| 外部链接 | 发出动作事件，由宿主处理 |
| 宏、程序和任意命令 | 不执行 |

## 4. 对象范围

| 对象 | 严格目标条件 |
|---|---|
| 形状和文字框 | 精确对象键对应唯一 DOM 元素 |
| 图片 | 同上，保留裁剪和静态变换 |
| 组合 | 组合本身有对象键 |
| 组合子对象 | 渲染器递归标记完整组合路径 |
| 表格 | 整体对象可定位；单元格动画不在第一版 |
| 图表 | 整体对象可定位；内部元素不在第一版 |
| SmartArt | 整体对象可定位；内部节点不在第一版 |
| 版式和母版对象 | 对象键包含来源，避免编号冲突 |

任何支持范围内的动画目标都必须唯一命中。缺失或重复时，该效果转为静态保留，不能落到名称相近对象。

## 6. 自动验收

### 6.1 解析和对象

- 支持范围内目标命中率必须为 100%；
- 每个目标只能命中一个元素；
- 时间节点编号、父子关系和条件与基准清单一致；
- 能力报告分母包含未识别节点；
- 未支持进入动画不能改变时间零可见状态。

### 6.2 时间和状态

- 对象、事件和单击顺序必须完全一致；
- 同一触发组的开始差不超过一个浏览器刷新周期；
- 500 毫秒以上效果的开始和结束误差不超过 `max(80ms, 持续时间的 5%)`；
- 暂停 500 毫秒期间中央时钟、属性和媒体不得推进；
- 重置和上一步后的稳定状态必须和从头求值得到的状态一致；
- 连续重播 10 次的事件记录必须一致。

### 6.3 属性和画面

- 时间零、25%、50%、75% 和结束状态检查属性值；
- 位移目标误差不超过 2 CSS 像素；
- 透明度误差不超过 0.05；
- 缩放比例误差不超过 1%；
- 旋转误差不超过 1 度；
- 静态对象的原始位置、层级和尺寸不得因动画包装变化；
- PowerPoint 截图用于人工确认裁剪、颜色、遮挡和切换画面。

数值门槛用于浏览器执行器自身和已人工标注的 PowerPoint 参数，不直接比较两套不同字体栅格化产生的整页像素。

### 6.4 操作和资源

- 快速下一步、上一步和跳页不会出现旧回调；
- 切换中最多保留一个待执行导航；
- 换文件后旧媒体、书签和动画不再触发；
- 组件卸载后没有活动动画、媒体、对象地址和事件监听；
- 隐藏页顺序导航和直接跳转规则正确；
- 自动播放被阻止时可以通过一次用户操作恢复。

## 7. 浏览器矩阵

正式发布必须在当前项目批准的 Chromium、Firefox 和 WebKit 版本运行：

| 能力 | Chromium | Firefox | WebKit |
|---|---|---|---|
| 加载和纵向静态预览 | 必须通过 | 必须通过 | 必须通过 |
| 基础时间顺序 | 必须通过 | 必须通过 | 必须通过 |
| 属性轨道 | 必须通过 | 必须通过 | 必须通过 |
| 页面切换 | 必须通过 | 必须通过 | 必须通过 |
| 媒体和书签 | 必须通过 | 必须通过 | 必须通过 |
| 全屏 | 用户操作下通过 | 用户操作下通过 | 用户操作下通过 |
| 近似 Morph | 单独记录 | 单独记录 | 单独记录 |

某浏览器不支持 CSS 独立变换时，渲染器适配器使用动画外层，不允许覆盖静态 `transform`。

## 8. 性能门槛

在批准测试机器上，对 43.4 MB、79 页业务文件进行三轮冷启动：

- 首个可见页面：不超过 2 秒；
- 已解析页面切换：不超过 500 毫秒；
- 下一步和暂停操作响应：不超过 100 毫秒；
- 当前页重置：不超过 200 毫秒；
- 连续重播 10 次后，释放页资源的内存增长不超过 20 MiB；
- 超过 50 MiB 的输入在解包前明确拒绝。

正式记录必须拆分来源读取、解包、建模、页面渲染、播放编译和媒体读取时间。

## 10. 发布暂停条件

出现以下任一情况，播放包不得发布：

- 支持目标存在缺失、重复或弱匹配；
- 时间节点树无法表达已批准素材；
- 同属性冲突仍产生矛盾最终状态；
- 暂停、重置、跳页或销毁留下旧回调；
- 未支持内容被计入严格执行；
- 三浏览器任一基础能力失败；
- 正式压缩包无法独立消费；
- 性能数据没有三轮固定环境证据。
