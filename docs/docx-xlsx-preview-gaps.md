# DOCX / XLSX 预览缺口记录

本文记录静态代码核对中发现的功能缺口及当前完成状态；任务依赖、验证命令和证据仍以稳定化路线图为准。

1. **已完成（P4-FIDELITY-CONTENT-02）**：DOCX 在引用所在页底部展示脚注正文，并在文档末页展示尾注正文；编号与正文可在正式 Demo 中直接核对。
2. **已完成（P4-FIDELITY-DOCX-01）**：DOCX Viewer 已能直接从公开模型展示批注、修订正文和页边卡片，并提供公开开关与 Demo 样例。
3. **已完成（P4-FIDELITY-CONTENT-02）**：DOCX Viewer 工具栏提供全文搜索、结果数、上一个/下一个和分页定位，命中段落有可见高亮。
4. **已完成（P4-FIDELITY-CONTENT-02）**：XLSX 单元格超链接和批注已进入公开工作表数据；网格显示超链接样式、受限协议跳转、批注角标和悬浮正文。
5. **已完成（P4-FIDELITY-CONTENT-02）**：XLSX 网格绘制数据条、色阶、图标集和折线/柱形/盈亏迷你图，主线程与 Worker 工作表快照使用同一公开数据。
6. **已完成（P4-FIDELITY-CONTENT-02）**：XLSX Viewer 增加只读形状与表单控件展示层，按工作表锚点、缩放、滚动和冻结区域定位，并显示形状文字、控件类型和选中状态。
7. **已完成（P4-FIDELITY-XLSX-WORKER-01）**：XLSX Worker 快照已带回图片资源和精确合并区域；正式 Demo 中 Worker 与主线程的图片位置、尺寸、合并绘制和选择锚点一致。
