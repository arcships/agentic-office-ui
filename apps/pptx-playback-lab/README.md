# PPTX 播放探索台

三段探索已经完成。本应用只用于复现 PPTX 原始播放机制和历史结论，不属于公开产品代码，也不能被正式包导入。

```bash
pnpm --filter=pptx-playback-lab dev
pnpm --filter=pptx-playback-lab typecheck
pnpm --filter=pptx-playback-lab build:lab
```

三段探索已完成，当前试验链路可以：

- 单次解包并保留 `PptxFiles`；
- 显示当前页 `p:timing` 和 `p:transition`；
- 按主时间序列提取多个单击批次；
- 区分单击时、与上一动画同时、上一动画之后；
- 将顶层幻灯片对象编号映射到渲染元素；
- 区分出现、淡入、消失和淡出；
- 执行缩放、擦除、强调和运动路径的试验动画；
- 按段落范围显示文字；
- 处理对象点击触发和视频播放、暂停；
- 从媒体关系中取出页面音频并实际播放；
- 执行淡化、推进、擦除和直接切换；
- 按页面计时自动换页；
- 下一动画、播放本页、暂停、继续和重置；
- 检查图表与 SmartArt 的内部目标；
- 识别 Morph 并尝试跨页对象匹配、位置和尺寸补间；
- 对旋涡等三维切换执行统一空间感近似；
- 读取媒体书签，并在视频到达书签时触发动画；
- 统计时间树深度、事件、重复、往返和缓动；
- 检测同一对象的并行动画属性冲突；
- 按显式名称、创建编号、文字、名称和几何评分匹配跨页对象；
- 将 `cTn` 保留为可追踪执行路径的时间节点树；
- 将同时开始的同对象效果合并为属性轨道；
- 记录解包、建模和逐页渲染耗时；
- 输出对象缺失和动画事件记录。

当前只验证 Chromium。强调、曲线路径、Morph、推进、擦除和三维切换仍包含近似画面；图表和 SmartArt 内部动画缺少语义编号；属性轨道只验证了同时开始的效果。正式版本需要按已验证的时间节点树、稳定对象身份和属性轨道重新实现，不能直接把本实验代码包装成产品。
